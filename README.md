# Random Tarot

Random Tarot is a web application for drawing tarot cards online.  
Users can generate a single-card reading or a three-card spread and view the results instantly.

The application is deployed on AWS and runs as a production web service.

---

## How to Use

1. Open the website in your browser.
2. Choose a draw mode:
   - **Single Card** – Draw one card.
   - **Three Cards** – Past, Present, Future spread.
3. Click **Draw**.
4. View the card name, orientation (Upright/Reversed), and image.
5. Repeat as desired.

No account or installation is required.

---

## Features

- Random tarot card selection
- Upright and reversed orientations
- Three-card spread mode
- Responsive web interface
- Cloud-hosted images (AWS S3)
- Production deployment on AWS
